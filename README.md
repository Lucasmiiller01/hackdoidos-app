# Hackadoidos App

### Installation

Hackadoidos App necessita da instalação do 
  -  Nodejs v10
  -  Yarn (Package Manager)
  -  Java JDK
  -  Android SDK
  -  Android Platform tools 28.0.3

#### Instalando as dependências

```sh
$ yarn
```

#### Copiando arquivo de configuração
```sh
$ cp .env .env.example
```

#### Rodando em desenvolvimento

```sh
$ react-native run-android
```
